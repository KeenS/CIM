NAME
----
ql-update - Update dist info and systems, or update quiclisp itself.

SYNOPSIS
--------
[verse]
ql update [client]

DESCRIPTION
-----------
Update installed systems. If target 'client' is given, update quicklisp itself.

EXAMPLES
--------
----
$ ql update client
----
