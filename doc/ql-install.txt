NAME
----
ql-install - Install given system.

SYNOPSIS
--------
[verse]
ql install <system>...

DESCRIPTION
-----------
Install given system(s) to $CIM_HOME/quicklisp. 'quickload' is also available as alias.

EXAMPLES
--------
* install clack
----
$ ql install clack
----
