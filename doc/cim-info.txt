NAME
----
cim-info - Display current state.

SYNOPSIS
--------
[verse]
'cim info'

DESCRIPTION
-----------
Display the information of current cim. It includes the environment variables that cim uses, what current and default lisp implementation is, what version of each implementation is used, and where system lisp is.

EXAMPLES
--------
* display info
----
$ cim info
----
