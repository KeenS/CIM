NAME
----
ql-list - List available/installed systems.

SYNOPSIS
--------
[verse]
'ql list' {remote | [local]}

DESCRIPTION
-----------

List installed systems. if 'remote' is given, list all the available systems. 'ql list local' is equivalent to 'ql list'.

EXAMPLES
--------
----
$ ql list remote
----
